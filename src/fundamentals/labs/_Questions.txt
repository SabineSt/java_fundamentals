1.) Using the command line, how do you:
    a. change directories?  cd ../ for up one; directoryname or path for downwards (also possible for up)
    b. list the contents of a folder?   ls (-al)(-Al)
    c. print the path of the current directory?   pwd
    d. move a file from one location to the another?   mv <file> <directory-name>
    e. copy a file from one location to another?   cp <file> <directory-name>
    f. create a file?   touch <name>
    g. create a folder?  mkdir <name>
    h. delete a file?   rm <filename>
    i. delete a folder?  rm -r <foldername>
    j. print the contents of a file?  cat <name>
    k. edit the contents of a file?  vim <name>
